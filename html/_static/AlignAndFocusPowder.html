This is a workflow algorithm that does the bulk of the work for time focusing diffraction data. This is done by executing several sub-algorithms as listed below.

  1. RemovePromptPulse (event workspace only)
  2. CompressEvents (event workspace only)
  3. CropWorkspace
  4. MaskDetectors
  5. Rebin or ResampleX if not d-space binning
  6. AlignDetectors
  7. If LRef, minwl, or DIFCref are specified:
    1. ConvertUnits to time-of-flight
    2. UnwrapSNS
    3. RemoveLowResTOF
    4. ConvertUnits to d-spacing
  8. Rebin if d-space binning
  9. DiffractionFocussing
  10. SortEvents (event workspace only)
  11. EditInstrumentGeometry (if appropriate)
  12. ConvertUnits to time-of-f
