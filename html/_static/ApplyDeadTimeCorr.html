Apply deadtime correction to each spectra of a workspace. Define:

 \({\displaystyle{N}}\) = true count
 \({\displaystyle{M}}\) = measured count
 \({\displaystyle{t_{dead}}}\) = dead-time
 \({\displaystyle{t_{bin}}}\) = time bin width
 \({\displaystyle{F}}\) = Number of good frames

Then this algorithm assumes that the InputWorkspace contains measured counts as a function of TOF and returns a workspace containing true counts as a function of the same TOF binning according to

\[N = \frac{M}{(1-M*(\frac{t_{dead}}{t_{bin}*F}))}\]
